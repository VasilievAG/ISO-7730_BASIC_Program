# Листинг программы на языке Бейсик к статье 

# Анализ неточностей и несоответствий в актуальной редакции стандарта ГОСТ Р ИСО 7730-2009

## [Вестник МГТУ. 2023. Т. 26, № 1. С. 45–56.]( https://vestnik.mauniver.ru/v26_1_n94/4_Vasileva_45-56.pdf)

## [eLIBRARY.ru (id 50468040)](https://elibrary.ru/item.asp?id=50468040)

## DOI: 10.21443/1560-9278-2023-26-1-45-56

В стандартах ISO 7730:2005 и ГОСТ Р ИСО 7730-2009 есть Приложение D с текстом программы на языке Бейсик. Этот текст, очевидно, набирался с распечатки, возможно, уже содержащей ошибки. Количество ошибок в тексте программы достаточно велико. Программа не только не запускалась, но даже восстановление кода программы оказалось не тривиальной задачей. Более того, в русской редакции стандарта были допущены ошибки в записи ключевых формул.

Все ошибки и неточности как русской, так и английской версий стандарта, перечислены в статье. Здесь же размещен файл с текстом программы на языке Бейсик. Программа адаптирована под диалект онлайн-интерпретатора [https://yohan.es/swbasic/]( https://yohan.es/swbasic/)
